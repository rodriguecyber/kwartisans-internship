# Artisan Internship Documentations

## Day 01: Git and Github basics

Day 01 of this internship, we have nothing to do except to learn Git and GitHub basics and create initial repository that will be used in the 20-days internship period.

## Day 02: PHP Data Generation & Algorithm Practice

Day 02 focused on building PHP data generation skills and algorithm problem-solving. A Machine Learning Engineer spends 80% of their time cleaning data.

### Tasks Completed:
-  **Resume Enhancement:** Research Senior Engineer resumes for technical keywords
- **Data Generator:** Created PHP script to generate 100 fake users (Name, Email, Age) in CSV format
- **LeetCode Challenge:** Solved "Palindrome Number" problem with multiple approaches
- **Documentation:** Daily progress tracking and code documentation

## Project Structure

```
kwartisans-internship/
├── REAME.md                    # Main project documentation
└── day02/                      # Day 02 internship tasks
    ├── generate.php            # PHP script to generate 100 fake users 
    └── DAILY_SUMMARY.md        # Detailed daily progress report
```

### File Descriptions:

#### `day02/generate.php`
- **Purpose:** Generates a CSV file with 100 fake users
- **Output:** `users.csv` with columns: Name, Email, Age
- **Features:** Random name generation, email creation, age assignment 


#### `day02/DAILY_SUMMARY.md`
- **Purpose:** Comprehensive daily progress documentation
- **Content:** Task details, code explanations, learnings, next steps
- **Status:** Day 02 completed successfully

## Technologies Used
- **PHP:** Data generation and algorithm implementation
- **Git:** Version control with feature branches
- **CSV:** Data export format
- **LeetCode:** Algorithm practice platform

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd kwartisans-internship
   ```

2. **Run PHP scripts:**
   ```bash
   cd day02
   php generate.php    # Generates users.csv

   ```

3. **View documentation:**
   - Open `/day02/DAILY_SUMMARY.md` for day 2 tasks
   - Check `REAME.md` for project overview
